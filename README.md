# Smart Study Planner Using Dynamic Programming

## Features
- Multi-user Login System
- Task Planner
- Weekly Calendar
- Dynamic Programming Task Optimizer
- Pomodoro Timer
- Analytics Dashboard
- Admin Panel
- Auto Task Rescheduling

## Technologies Used
- Python
- Flask
- SQLite
- HTML/CSS/JavaScript
- FullCalendar
- Chart.js

## Dynamic Programming
This project uses the 0/1 Knapsack Algorithm to generate optimized study schedules based on task priority and available study hours.
